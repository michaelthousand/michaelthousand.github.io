String Searcher

The following application searches a specified text file for a string. The user passes the string and file name as command line arguments and the program returns the number of occurences.

As you can see below, the program checks for improper arguments and can handle some exceptions that may occur. If there are no errors, it prints out the number of occurences in the file.
